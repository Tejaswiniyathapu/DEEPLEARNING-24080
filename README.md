# Deep Learning Lab

This repository contains the implementation of Deep Learning laboratory experiments completed as part of the **Deep Learning (22AIE304)** course. The experiments demonstrate the progression from a simple Perceptron to Deep Neural Networks (DNNs) using TensorFlow and Keras.

## 📚 Labs Included

### Lab 1: Perceptron Learning
- Perceptron implementation from scratch
- AND and OR gate classification
- Decision boundary visualization
- Perceptron learning algorithm

### Lab 2: Multi-Layer Perceptron (MLP)
- XOR gate classification
- Forward propagation
- Backpropagation
- Hyperparameter tuning
- Activation function comparison (Sigmoid vs ReLU)

### Lab 3: Hyperparameter Optimization
- Learning rate tuning
- Batch size analysis
- Dropout regularization
- Grid Search / Random Search
- Model performance comparison

### Lab 4: Deep Neural Network (DNN)
- MNIST handwritten digit classification
- Data preprocessing
- Model training and evaluation
- Confusion matrix
- Error analysis
- Performance metrics
